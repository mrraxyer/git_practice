Esta aplicacion crea una pagina web sencilla con el objetivo de practicar el uso de un controlador de versiones (git/github) en equipo. 

## Instalación

1. Clona el repositorio:
   `https://github.com/mrraxyer/git_practice.git`

## Flujo de trabajo seguido

1. Creacion del repositorio 
2. Asignacion de tareas a cada integrante, comunicacion de orden.
3. Seguimiento de funciones (especificada en el docs) 
    3.1 Primera funcionalidad. Concepto de equipo
    3.2 Segunda funcionalidad. Navbar de la pagina
    3.3 Tercer funcionalidad. Footer de la pagina
    3.4 Cuarta funcionalidad. Agregando estilo a la pagina
    3.5 Quinta funcionalidad. Correccion de errores en el trabajo
4. Documentacion de actividades
